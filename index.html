Alexa Tone Analyzer is an Alexa Skill that allows the user to orally journal. The user dictates a journal "entry" to Alexa who then calls on IBM Watson to obtain a tonal analysis. The user's entry and tonal analysis is stored on Amazon's DynamoDB along with metadata.
